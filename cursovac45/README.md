# Sneakershop

Простой Laravel-проект (магазин кроссовок) — исходный код загружен в этот репозиторий.

Краткие инструкции:

- Установите зависимости:

```bash
composer install
npm install
npm run build
```

- Настройте окружение: скопируйте `.env.example` в `.env` и заполните параметры базы данных.

- Выполните миграции и сиды (если нужно):

```bash
php artisan migrate --seed
```

- Запустите локальный сервер:

```bash
php artisan serve
```
